Before beginning to write the HTML code for the website we did some preparation. This firstly included doing all the labs, as well as making a prototype digitally on the app Adobe XD where we were able to design the layout. This gave us a clear idea of which color pallet to use as well as determining the position of elements such as pictures, paragraphs, and headlines. We also wrote all of the text belonging to each page in a document before copy-pasting it into the HTML code. This helped us know what would be in italic, bold, when to use abbreviations, new paragraphs ect. If we had a problem understanding a certain part of the code we would look up on the website W3schools, for instance when making the navigation bar. We chose to have the CCS properties on each page index so that it was clearer for us what style elements belonged to what page. In the CCS part of the code we imported animations, fonts and determined the position of elements such as the navigation bar which we made “sticky” in order to be accessible at all times on the website.
